This is a webpage to add,delete and expand details of each rows in a table.
HTML tags used: <table>,<nav>,<script>,<button> are used to create layout of the web page.
In function addStudent: var Keyword, onclick property and in-built functions- getElementsByClassName, getElementById to access the table and alert, querySelectorAll, insertCell,appendChild, 
setAttribute,createElement are used to add each row to that table.
In selectStudent function: checked property, getElementById function is used to select each row and if- else statement is used for CSS.
In showDetail function: isVisible property, getElementById function and if-else statemnt to display details row.
getElementById, alert function and display properties are used for deleteStudent function.
